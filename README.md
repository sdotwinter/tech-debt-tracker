# Tech Debt Tracker

[![Sponsor](https://img.shields.io/badge/Sponsor-GitHub%20Sponsors-ea4aaa?logo=githubsponsors)](https://github.com/sponsors/sdotwinter)

A Python CLI to track, score, and prioritize technical debt.

## Usage
```bash
python3 main.py add "Refactor auth middleware" --area backend --impact 8 --effort 5 --risk 7 .
python3 main.py list .
python3 main.py report . --json
python3 main.py close 1 .
```
