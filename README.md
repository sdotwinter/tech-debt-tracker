# Tech Debt Tracker

[![Sponsor](https://img.shields.io/badge/Sponsor-GitHub%20Sponsors-ea4aaa?logo=githubsponsors)](https://github.com/sponsors/sdotwinter)

Track and prioritize technical debt in engineering projects.

## Usage
```bash
python3 main.py add --title "Refactor auth middleware" --area backend --impact 8 --effort 5 --risk 7
python3 main.py list
python3 main.py report --format md
python3 main.py report --format json
```
